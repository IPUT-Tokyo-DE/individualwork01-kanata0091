# Avoid the Explosion

## ゲームの内容
画面上から落下してくる三角や四角が爆発するため、それをひたすらよけるゲームです。Player（Circle）の操作を入力した向きと逆にしています。
このゲームは完成していません。
課題提出日に自分の管理不足によりデータが消えてしまったため、できるところまでやりました。

![画面イメージ](docs/image01/game_image01.png)


## 操作説明
Playerは矢印キーで操作します。
入力した方向と逆方向に動きます。
